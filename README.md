# 每日主题 - 随心情变化的动态界面

一个基于时间和心情的智能主题切换网页，提供丰富的交互体验和视觉效果。

## ✨ 主要功能

### 🎨 主题系统
- **自动模式**: 根据时间自动切换主题（清晨、午后、黄昏、夜晚）
- **心情主题**: 开心、平静、活力、浪漫等主题
- **随机主题**: 每天随机选择不同主题
- **平滑过渡**: 优雅的主题切换动画

### 🎵 音乐播放器
- 背景音乐播放控制
- 音量调节
- 多首音乐切换
- 与主题同步的音乐体验

### 🌤️ 实时天气
- 基于地理位置的天气信息
- 天气图标和温度显示
- 天气描述和位置信息
- 手动刷新功能

### ⏰ 倒计时功能
- 自定义重要日期倒计时
- 实时倒计时显示（天、时、分、秒）
- 本地存储保存设置
- 倒计时结束提醒

### 🎤 语音控制
- 语音命令识别
- 支持主题切换命令
- 音乐播放控制
- 粒子效果触发

### 🎮 互动效果
- **粒子效果**: 点击释放彩色粒子
- **颜色变换**: 卡片颜色动态变化
- **动画播放**: 元素脉冲和发光效果
- **打字机效果**: 文字逐字显示动画
- **庆祝彩带**: 彩色彩带飘落效果
- **彩虹模式**: 全屏彩虹色彩变换

### 🎯 3D交互
- 3D卡片翻转效果
- 鼠标悬停3D变换
- 立体视觉体验

## 🚀 使用方法

### 键盘快捷键
- `1-8`: 快速切换主题（1-清晨, 2-午后, 3-黄昏, 4-夜晚, 5-开心, 6-平静, 7-活力, 8-浪漫）
- `0`: 返回自动模式
- `r`: 随机主题
- `p`: 释放粒子效果
- `c`: 变换颜色
- `a`: 播放动画
- `t`: 打字机效果
- `f`: 庆祝彩带
- `b`: 彩虹模式

### 触摸手势
- **向上滑动**: 切换到随机主题
- **向下滑动**: 切换到自动模式

### 语音命令
- "切换主题" - 随机主题
- "清晨模式" - 切换到清晨主题
- "夜晚模式" - 切换到夜晚主题
- "播放音乐" - 播放背景音乐
- "释放粒子" - 触发粒子效果

## 🔧 自定义

可以通过修改CSS变量来自定义主题色彩：

```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    --accent-color: #your-color;
    --background-color: #your-color;
    --surface-color: #your-color;
    --text-primary: #your-color;
    --text-secondary: #your-color;
}
```
